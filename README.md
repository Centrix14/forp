# forp - FORk Processor
forp is a simple, homoiconic, two-dimensional, programming language

## example
```
;
	this is comment
	"Hello, world!" programm

print
	Hello, world!
	.n

;
	The Pythagorean theorem

print
	+
		*
			3
			3

		*
			4
			4	

	.n
```

for more examples see `samples/`

## compile
```
make rel
```
